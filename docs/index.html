SynthDiD.jl

Julia implementation of Synthetic Difference-in-Differences with rendered vignettes for the California Proposition 99 example, plotting workflows, and replication of the paper’s main results.

  • Introduction — California Prop 99 walkthrough and estimator overview.
  • More on Plotting — plot customization, effect curves, and weight displays.
  • Paper Results — replication-style outputs corresponding to the paper.

Repository: github.com/xiangao/SynthDiD.jl
